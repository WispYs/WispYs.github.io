3. v-show 控制元素是否显示

4. v-if 与 v-show 的不同

v-if 是真实的条件才会渲染，如果初始条件是假的则不会做任何处理，只有在条件第一次变为真时才开始局部编译

v-show 的元素会始终渲染并保持在 DOM 中，v-show 是简单的切换元素的 CSS 属性 display 。

一般来说， v-if 有更高的切换消耗而 v-show 有更高的初始渲染消耗。因此，如果需要频繁切换使用 v-show 较好，如果在运行时条件不大可能改变则使用 v-if 较好。
